# Mentor_Mentee_Portal
# 🚀 Mentor Mentee Portal (MentorHub)

A full-stack web application built using Flask that enables seamless interaction between mentors and mentees through task management, submissions, grading, and performance tracking.

---

## 📌 Features

### 👤 Authentication
- User Registration & Login
- Role-based access (Mentor / Mentee)

### 📊 Dashboard
- Mentees:
  - View average, max, min scores
  - Track progress with charts
- Mentors:
  - View total mentees
  - Monitor overall performance

### 📚 Task Management
- Mentors can assign tasks with deadlines
- File upload support for tasks
- Mentees can view assigned tasks

### 📥 Submission System
- Upload assignments (PDF, images, docs)
- Track submission status:
  - Pending
  - Submitted
  - Graded

### 📝 Grading System
- Mentors can:
  - Assign scores (/100)
  - Provide feedback
- Automatic status updates

### 📈 Performance Analytics
- Graphical insights using Matplotlib
- Individual and comparative performance tracking

### 💬 Messaging System
- Chat between mentor and mentee
- Contact-based conversation UI

### 📂 File Handling
- Secure uploads
- In-browser PDF viewing

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** HTML, CSS, Jinja2
- **Libraries:**
  - Pandas
  - NumPy
  - Matplotlib
